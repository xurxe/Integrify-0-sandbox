10. Terminal:

	git status —> get current status (note: use this often!)
	git add index.html —> stages file
	git add README.rtf
	git commit -m “Initial commit” —> commits file locally
	git log —> gets log

	git add README.rtf
	git commit -m “Second commit”
	git log
	git log --oneline —> neat little summary of the log
